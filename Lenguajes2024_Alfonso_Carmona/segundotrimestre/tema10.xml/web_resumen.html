Resumen de XML

TEMA 10

XML es un lenguaje de marcado de propósito general que es compartir datos a traves de diferentes sistemas, como internet.

Caracteristicas Principales

  • Lenguaje de estructura libre. Las etiquetas se definen.
  • Consta de dos partes: Prólogo y cuerpo.
  • Usa espacios de nombres para evitar ambigüedades.
  • Su sintaxis es como la de HTML
  • Debe cumplir las reglas w3c para estar bien formado
  • Los documentos se validan
