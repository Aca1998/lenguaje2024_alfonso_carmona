¿Para que sirve Xpath?

TEMA 12

Sirve para extraer datos , para tener acceso a todos llos nodos de XML, para visualizar elementos y para hacer calculos.

Tipos de Nodos

  • Raiz, Descendientes, Ascendendtes, Hermanos.
  • Raiz: El nodo raiz siempre es /.
  • Elemento, Atributo, Texto.

Notas sobre el eje Xpath

  • nombre_nodo: Seleciona todos los nodos con ese nombre
  • / : Selecion de nodos desde el nodo raiz
  • // : Seleciona nodos desde el nodo actual
  • . : Seleciona el nodo actual
  • .. : Seleciona el padre del nodo actual
  • @ : Seleciona atributos
  • | : Permite indicar varios caminos

    • Sintaxis Xpath

    Una expresion Xpath es una cadena de texto que localiza nodos de un arbol XML

    • Notas sobre seleccion de nodos

    • node (): Seleciona todos los hijos (Elementos o texto).
    • text () : Seleciona unicamente el texto contenido y no muestra las etiquetas.
    • * : Seleciona todos los hijos (solo elementos).
    • @* : Seleciona todos los atributos del nodo.

    Formatos de intercambios de datos

      JSON : JavaScript Object Notation
  • Formato ligero para el intercambio de datos.
  • Basado en pares clave-valor y arrays.
  • La extensión es: .json
  • Compatible con la mayoría de lenguajes de programación.

    • Formatos de intercambios de datos

      CSV: Comma-Separated Values
  • Formato tabular donde los datos están separados por comas.
  • Usado para almacenar datos simples en texto plano.
  • La extensión es: .csv
  • Sencillo: Fácil de crear y leer.

    • Formatos de intercambios de datos

      YAML: YAML Ain't Markup Language
  • Utiliza indentación para organizar los datos jerárquicamente.
  • Más fácil de leer y escribir que XML o JSON.
  • La extensión es: .yml o yaml
  • Flexible: Permite representar estructuras de datos complejas.
